Front-End

Bagian front-end dari sebuah website adalah bagian yang dapat dilihat langsung oleh user pada layar. User juga bisa langsung berinteraksi pada bagian ini. Front-end berhubungan dengan UIUX yang mengatur tampilan website semenarik mungkin bersamaan dengan merancang pengalaman pengguna untuk memberikan kesan nyaman ketika berinteraksi. Bagian ini dibuat menggunakan HTML, CSS, dan JavaScript.

Back-End

Back-end adalah bagian belakang layar dari sebuah website. Bagian dalam mengatur fungsionalitas website agar bekerja dengan baik. Bagian ini lebih berkutat dengan database dan manajemen konten di suatu website namun juga bertanggung jawab memastikan keamanan sebuah web. Bahasa pemograman untuk back-end development diantaranya adalah PHP, Ruby, Python, dan lainnya.
